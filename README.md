# Task_management_Sprint_2
This is a simple and responsive Task Management application built using HTML, CSS, JavaScript, and Bootstrap. The application allows users to add new tasks with details such as task name, description, assigned person, due date, and status. It includes form validation, dynamic task rendering, priority selection, and status-based styling for better user experience.

Through this project, I focused on clean UI design, proper validation handling, DOM manipulation, and writing secure JavaScript code.
